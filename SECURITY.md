# Security Policy

## Reporting a Vulnerability

To report a security vulnerability, please use GitHub's
[Security Advisories](https://github.com/Legolas91/claude-code-api-proxy/security/advisories/new)
feature — **do not open a public issue**.

Reports are reviewed on a best-effort basis.

## Scope

- API key forwarding and credential handling
- CORS policy bypass
- Request injection via crafted payloads
- Information disclosure via debug logs

## Out of Scope

- Vulnerabilities in the upstream providers (OpenAI, OpenRouter, Ollama, etc.)
- Issues requiring physical access to the host machine
